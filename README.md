# Client-Device-Preferences-Dashbaord
Built a Tableau dashboard to track device type (android vs iOS) and app version adoption, helping stakeholders understand user behavior and drive platform optimization.
Concepts: Data Visualization, User Segmentation, Trend Analysis
Implementation: Tableau Desktop, MySQL, SQL, Excel, Tableau Server
